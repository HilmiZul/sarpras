## Setup


```bash
# npm
npm install

# pnpm
pnpm install

```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

```

## Production

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm build

```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm preview

```
